## 🔄 HOW IT WORKS

### **Discord Bot Flow:**
```
User sends message
    ↓
Discord bot receives message
    ↓
Calls: core.process_message(user_message, emotion_data=None)
    ↓
Full 19-stage cognitive pipeline runs:
    1. Perception Layer
    2. Enhanced Semantic Understanding
    3. Conflict Detection
    4. Psyche Engine Update (Neurochemicals FIRST, then mood)
    5. Embodiment State Update
    6. Memory System Update
    7. Temporal Awareness
    8. Reciprocity Ledger Update
    9. Parallel Life Awareness
    10. CPBM Learning (with engagement calculation)
    11. Personality Layer Updates
    12. Relationship Phase Evaluation
    13. Memory Selection (Stochastic)
    14. Theory of Mind
    15. QMAS Debate (with Creativity Engine)
    16. Two-Stage LLM Reasoning
    17. Creativity Engine
    18. Self-Narrative Generation
    19. Message Sequence Planning
    20. State Saving
    ↓
Bot generates response using all cognitive state
    ↓
Bot sends response to Discord
```

**Every single stage runs!** The Discord bot uses the exact same `CognitiveCore.process_message()` function.
