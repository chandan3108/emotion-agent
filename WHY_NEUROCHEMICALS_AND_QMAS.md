## Neurochemicals: The Biological Foundation

### How Real Emotions Work

**Real emotions aren't just mood vectors.** They're driven by:
- **Dopamine (DA)**: Reward, motivation, novelty → happiness, excitement
- **Cortisol (CORT)**: Stress response → anxiety, tension
- **Oxytocin (OXY)**: Bonding, trust → affection, warmth
- **Serotonin (SER)**: Mood stabilizer → contentment, confidence
- **Norepinephrine (NE)**: Alertness, focus → energy, attention

### Why This Matters

**Without neurochemicals:**
- Mood changes feel arbitrary
- Emotions don't have biological grounding
- Behavior isn't driven by underlying mechanisms

**With neurochemicals:**
- Events trigger realistic hormonal responses
- Mood is **driven by** neurochemicals (not just set directly)
- Behavior emerges from biological state
- **More realistic** - this is how humans actually work

### The Flow

```
Event (conflict) 
  → Neurochemical cascade (CORT↑, OXY↓, DA↓)
    → Mood changes (stress↑, affection↓, happiness↓)
      → Behavior changes (withdrawn, defensive, shorter messages)
```

**This is authentic.** Not just "if conflict then mood = stressed" - it's a biological cascade.
